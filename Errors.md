1. How to push your code to GitHub
    i. Using the command line
      a.git init
      b.git add .
      c.git commit -m "Your commit message"
      d.git remote add origin <repository_url>
      e.git push -u origin master
      if you have already a readme file in your repository, you can use the following command to pull the changes before pushing your code:

      f.git pull origin master --allow-unrelated-histories
      if not working the f. command
        git config pull.rebase false
